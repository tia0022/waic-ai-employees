# 协同场景 H：TikTok 爆品追单（Viral Product Scaling）

> **视角**：正面（爆发机会） | **触发**：TikTok Shop 单品 24h 内销量突破 10x 均值
> **版本**：V0.1（2026-05-25）
> **步骤经济性**：每个小明 ≤ 3 步 | 全场景 ≤ 12 步 | HITL ≤ 3

---

## 〇. 场景定位

| 项 | 内容 |
|---|---|
| 老板诉求 | "TikTok 上视频爆了，库存只够卖 3 天——能不能 72 小时内补上？别让流量白白流走" |
| 品类 | 家居小件 / 创意日用品（客单价 $15-$50，轻量可空运） |
| 市场 | US TikTok Shop 为主 → 同步 Amazon / Shopify 独立站 |
| 时间窗口 | 72 小时黄金期（爆品流量衰减半衰期 ~3 天） |
| 正面叙事核心 | **在流量爆发的 72h 窗口内，5 个小明协同完成"从发现→追单→合规→发运→放量"全链路** |

---

## 一、触发条件

```
IF TikTok_Shop_SKU.sales_24h > 10 × avg_daily_sales
AND inventory_days < 5
THEN trigger_scenario_H
```

---

## 二、执行时间轴（72 小时）

| 时段 | 主导小明 | 任务 |
|---|---|---|
| T+0h → T+2h | 经营分析 + 营销 | 确认爆品趋势真实性 + 流量来源分析 |
| T+2h → T+8h | 物流 + 合规 | 追单可行性评估 + 空运报价 + 合规校验 |
| T+8h → T+72h | 营销 + 物流 + 经营评估 | 广告追投 + 补货发运 + ROI 实时监控 |

---

## 三、步骤详情

### Phase 1：爆品确认（T+0h → T+2h）— 3 步

| Step | 小明 | 动作 | HITL |
|---|---|---|---|
| H-1 | 经营分析 (COA) | 拉取 TikTok Shop 实时数据 + Amazon 同 ASIN 趋势 + Google Trends 验证 → 判断"真爆"vs"一次性" → 输出 72h 销量预测 | — |
| H-2 | 营销 (SCORE) | 分析爆款视频特征（达人/话题/音乐）→ 识别可复制的流量入口 → 输出"追投策略建议" | — |
| **HITL-1** | — | **爆品追单启动卡**（决策人=运营总监；附 72h 销量预测 + 追单成本 vs 预期利润 + 风险提示） | **Go/No-Go** |

### Phase 2：追单准备（T+2h → T+8h）— 4 步

| Step | 小明 | 动作 | HITL |
|---|---|---|---|
| H-3 | 物流 | 联系供应商确认产能 → 询价空运（深圳→LAX 48h 快线）→ 计算补货成本 | — |
| H-4 | 合规 | 校验爆品 HS 编码 + 目标国认证（如 CPSC/ASTM 安全标准）→ 确认无合规阻塞 | — |
| H-5 | 经营评估 (DE) | 综合空运成本 + 预期毛利 → 输出追单 ROI（含不同补货量的盈亏平衡点） | — |
| **HITL-2** | — | **追单执行审批卡**（决策人=CEO；附补货量 × 成本 × 预期回报 + 空运时效） | **决策** |

### Phase 3：执行与放量（T+8h → T+72h）— 4 步

| Step | 小明 | 动作 | HITL |
|---|---|---|---|
| H-6 | 物流 | 下单追货 + 安排空运 + 实时追踪 → 预估到仓时间同步给营销 | — |
| H-7 | 营销 (SCORE) | 追投 TikTok 达人合作 + 提升 Amazon 广告 bid + 独立站引流 → 多渠道放量 | — |
| H-8 | 经营分析 (COA) | 实时监控 GMV/ACoS/库存消耗速度 → 如果热度衰减提前 → 触发减量信号 | — |
| **HITL-3** | — | **72h 战况复盘卡**（决策人=运营总监；附实际 ROI vs 预测 + 是否继续补货） | **决策** |

## 四、步骤经济性核算

| 小明 | 步骤数 | 覆盖 Phase |
|---|---|---|
| 经营分析 (COA) | 2 | Phase 1 + Phase 3 |
| 营销 (SCORE) | 2 | Phase 1 + Phase 3 |
| 物流 | 2 | Phase 2 + Phase 3 |
| 合规 | 1 | Phase 2 |
| 经营评估 (DE) | 1 | Phase 2 |
| **合计** | **8 步** | 3 Phase + 3 HITL |

> 注：8 步是所有正面场景中最少的——因为 TikTok 爆品窗口极短，每一步都必须在小时级完成，步骤越少越好。
