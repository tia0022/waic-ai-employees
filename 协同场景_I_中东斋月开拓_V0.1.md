## 一、为什么是斋月 + 中东？

| 数据点 | 说明 |
|---|---|
| GCC 电商规模 | 2025 年 ~$50B，年增 25%+ |
| 斋月消费增长 | 斋月期间消费支出 +30-50%（夜间购物文化） |
| 平台机会 | Noon.com / Amazon.ae / Namshi 对中国卖家开放 |
| 物流基建 | 中国→迪拜海运仅 12-15 天（比到美国短） |
| 合规门槛 | 相对欧美较低（无 CPSC/FCC），但有清真认证 + 阿联酋标准 ESMA |

## 六、中东市场特殊考量

| 维度 | 注意事项 | 负责小明 |
|---|---|---|
| 文化合规 | 产品图片/描述不得含猪/酒/暴露形象；颜色偏好（金/绿/蓝） | 合规 + 营销 |
| 时区运营 | UAE = UTC+4；斋月消费高峰在 Iftar (日落) 和 Suhoor (凌晨) | 营销 |
| 支付 | Tamara/Tabby 先买后付流行；Apple Pay/Samsung Pay 覆盖率高 | — |
| 退货政策 | UAE 法律要求 14 天无条件退货（比 Amazon US 更严） | 合规 |
| 物流节奏 | 斋月期间海关清关速度可能 -30%（政府机构缩短工时） | 物流 |
| 本地化 | RTL（从右到左）排版；阿拉伯语 listing 质量直接影响转化 | 营销 |
