## 二、每个场景的 WAIC 现场适配评估

### 场景 A：全渠道利润拯救与爆款全球扩张

| 评估维度 | 评分 | 说明 |
|---|---|---|
| 故事一致性（老板下一句话→交付物） | ★★★★★ | "在不增加风险下提升利润 + 找下一爆款" 是 CEO 最常说的真实诉求 |
| 五位 Agent 全员上场 | ★★★★★ | 5 个全用上，体现 Agent Mesh |
| HITL 真实感 | ★★★★ | 3 个决策卡（广告预算/贷款额度/空运比例），每个都贴近真实 CEO 权衡 |
| 现场视觉冲击 | ★★★★ | 决策卡可视化 + CEO 行动清单打印瞬间出片 |
| 现场可演示时长 | 5~8 分钟（180s 时间轴 + 老板讲解） | 偏长，需脚本压缩 |
| **风险点** | – | 5 个 Agent 同屏调度，UI 稍复杂；需确保现场不卡顿 |
| **推荐用途** | **主 Demo（必选）** | 完整体现"老板说一句话→五位小明协同→打印 CEO 行动清单"的旗舰叙事 |

### 场景 B：供应链重大危机处置与产业金融联动

| 评估维度 | 评分 | 说明 |
|---|---|---|
| 故事一致性 | ★★★★★ | "ETA 延迟 7 天 + FBA 断货 + 资金缺口" 是真实发生频次最高的危机 |
| 五位 Agent 全员上场 | ★★★★★ | 5 个全用上 |
| HITL 真实感 | ★★★★★ | 物流方案选项（海运/空运/混合）现场让老板拨拨条选择，沉浸感最强 |
| 现场视觉冲击 | ★★★★★ | 三套方案 ROI 对比图 + 银行接口推送动画 + DE 报告生成 |
| 现场可演示时长 | 8~10 分钟（18min 时间轴需压缩为 5min） | 需要做剪辑压缩 |
| **风险点** | – | 时间轴较长，需要压缩；金融接口部分如果接不通会显得空 |
| **推荐用途** | **主 Demo（必选）** | 把"产业 + 金融"联动讲清楚的最佳载体——这是 AXON GLOBAL 与同行最大的差异点 |

### 场景 C：法国市场 VAT/EPR 风险拦截与合规增长

| 评估维度 | 评分 | 说明 |
|---|---|---|
| 故事一致性 | ★★★★ | 增长前先合规，但故事张力不如危机型场景 |
| 五位 Agent 全员上场 | ★★★ | 4 个，没用上经营评估的核心能力 |
| HITL 真实感 | ★★★ | 决策卡偏"配置型"，不如选方案有体验感 |
| 现场视觉冲击 | ★★★ | 风险卡片 + 合规话术对比，偏静态 |
| 现场可演示时长 | 3~5 分钟 | 可控 |
| **风险点** | – | 偏 B2B 内行的故事，普通观众感受不深 |
| **推荐用途** | **辅助 Demo（备选）** 或 **演讲 Slide 案例** | 单独 Demo 略弱；建议**作为场景 A 的"嵌入子场景"**——A 中的"法国扩张前合规预检"那段直接展开讲，更高效 |

### 场景 D：爆款达人资产沉淀与供应链联动

| 评估维度 | 评分 | 说明 |
|---|---|---|
| 故事一致性 | ★★★★ | "达人爆了→撞单→样品乱→断货" 这个故事观众秒懂 |
| 五位 Agent 全员上场 | ★★★ | 4 个，合规略显边缘 |
| HITL 真实感 | ★★★★ | 报价/样品规模/预算都是真实决策点 |
| 现场视觉冲击 | ★★★★ | TikTok 视频 embed + 达人画像 + 撞单防御卡，年轻观众喜欢 |
| 现场可演示时长 | 3~5 分钟 | 可控 |
| **风险点** | – | 与 A 场景的"法国爆款扩张"部分高度重合，避免重复 |
| **推荐用途** | **辅助 Demo（备选）** | 如果展会受众偏卖家、运营、达人 MCN，该场景比 C 更有共鸣；建议**与 C 二选一**作为场景 A 的补充 |

### 场景 E：多平台经营下滑后的 CEO 经营复盘与组织派单

| 评估维度 | 评分 | 说明 |
|---|---|---|
| 故事一致性 | ★★★★★ | "生意变差，告诉我真正原因" 比"利润修复"更老板视角 |
| 五位 Agent 全员上场 | ★★★★★ | 5 个全员，且四方独立诊断+主导整合是叙事亮点 |
| HITL 真实感 | ★★★ | 决策卡偏"是否同意行动清单"，没有 A/B 的方案选择那么戏剧 |
| 现场视觉冲击 | ★★★★★ | "四个部门各执一词→主导整合统一答案" 是最容易让观众产生顿悟的瞬间 |
| 现场可演示时长 | 4~6 分钟 | 可控 |
| **风险点** | – | 与 A 场景同为内部分析师主导，叙事路径会显重复 |
| **推荐用途** | **主 Demo 候选（与 A 二选一）** | 比 A 更"管理层视角"，比 A 更有"AI 解开人类盲区"的戏剧性；如果选 E 替代 A，整体观感更上层 |

## 三、推荐组合（V0.1 建议）

### 方案一：保守组合（**推荐**）

| 序 | 场景 | 时长 | 角色 |
|---|---|---|---|
| 1 | **B 供应链危机+金融联动** | 5~6 min | 主 Demo · 产融联动旗舰 |
| 2 | **A 利润拯救+爆款扩张** | 5~6 min | 主 Demo · CEO 行动清单旗舰 |
| 3 | **D 爆款达人** | 3 min | 现场互动小演示 · 给营销/MCN 观众 |

**总时长 ≈ 13~15 分钟**，展位每轮观众一次完整体验，5 位小明全部出场两次，HITL 决策卡多次出现。

### 方案二：创新组合

| 序 | 场景 | 时长 | 角色 |
|---|---|---|---|
| 1 | **E 经营下滑根因诊断** | 5 min | 主 Demo · 给观众"原来 AI 能看穿部门盲点"的顿悟感 |
| 2 | **B 供应链危机** | 6 min | 主 Demo · 产融联动 |
| 3 | C 或 D（按受众选）| 3 min | 辅助 |

**风险**：E 比 A 抽象一点，对非业内观众解释成本略高。

### 方案三：极简组合

只演 **A + B**，13 分钟一轮。最稳，但少了花样。
